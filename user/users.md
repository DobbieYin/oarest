#登录
>接口描述

| 接口名称 | *登录* |
| -- | -- |
| **接口地址** | */users/login* |
| **请求方式** | <mark>GET</mark> |
| **数据格式** | <code>JSON</code> |


##请求参数
[<公共传入参数>](../README.md)  

|编码|名称|类型|必输|说明|默认值|
|:---|:---|:---|:--:|:---|:-----|
|username|用户名|<code>string</code>|是|暂无|无|
|password|密码|<code>string</code>|是|暂无|无|


##返回参数
[<公共返回参数>](../README.md)

|编码|名称|类型|必输|说明|默认值|
|:---|:---|:---|:--:|:---|:-----|
|data|返回数据|<code>object</code>|是|暂无|无|

参数项：data

|编码|名称|类型|必输|说明|默认值|
|:---|:---|:---|:--:|:---|:-----|
|userid|用户ID|<code>string</code>|是|暂无|无|
|access_token|访问令牌|<code>string</code>|是|暂无|无|
