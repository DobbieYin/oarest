#登录
>接口描述

| 接口名称 | *登录* |
| -- | -- |
| **接口地址** | */users/login* |
| **请求方式** | <mark>GET</mark> |
| **数据格式** | <code>JSON</code> |


##请求参数
[<公共传入参数>](../README.md)  

|编码|名称|类型|必输|说明|默认值|
|:---|:---|:---|:--:|:---|:-----|
|username|用户名|<code>string</code>|是|暂无|无|
|password|密码|<code>string</code>|是|暂无|无|
