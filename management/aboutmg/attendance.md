| 接口名称 | *指定日期查询考勤记录* |
| -- | -- |
| **接口地址** | */permis* |
| **请求方式** | <mark>GET</mark> |
| **数据格式** | <code>JSON</code> |


##请求参数
[<公共传入参数>](../README.md)  

|编码|名称|类型|必输|说明|默认值|
|:---|:---|:---|:--:|:---|:-----|
|date|用户ID|<code>string</code>|否|当flag为2时，需要指定该项参数|当前用户|


##返回参数
[<公共返回参数>](../README.md)

|编码|名称|类型|必输|说明|默认值|
|:---|:---|:---|:--:|:---|:-----|
|data|权限数组|<code>array</code>|是|暂无|无|
